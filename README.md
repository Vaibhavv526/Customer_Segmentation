# 🚀 Customer Segmentation using K-Means

This project performs **customer segmentation** using Machine Learning (K-Means Clustering) and provides an interactive **Streamlit web application** for real-time predictions and insights.

---

## 📌 Project Overview

Customer segmentation helps businesses understand their customers better by grouping them based on behavior and attributes such as:

- Age
- Income
- Spending behavior
- Purchase patterns

This project:
- Applies **K-Means clustering**
- Uses **feature scaling**
- Visualizes clusters using **PCA**
- Deploys a **Streamlit app** for prediction

---

## 🧠 Machine Learning Approach

- Algorithm: **K-Means Clustering**
- Scaling: **StandardScaler**
- Visualization: **PCA (2D Projection)**
- Evaluation: **Elbow Method**

---

## 📊 Features Used

- Age  
- Income  
- Total Spending  
- Number of Store Purchases  
- Number of Web Purchases  
- Number of Web Visits per Month  

---

## 🎯 Customer Segments

| Cluster | Segment Name |
|--------|-------------|
| 0 | Budget Customers |
| 1 | Premium Customers |
| 2 | Loyal High Spenders |
| 3 | Regular Customers |
| 4 | Occasional Buyers |
| 5 | VIP Customers |

---

## 🖥️ Streamlit Web App

The app allows users to:
- Input customer details
- Predict customer segment
- Get business recommendations
